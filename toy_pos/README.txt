common_.py : 공용으로 사용 될 변수(list)이 들어가는 파일 
            ex)상품목록, 판매목록...

print_.py : 공용으로 사용 될 print들을 담아 둔것
             (main에서 지저분하지 않게 따로 모듈화)

product_.py : 상품관리의 기능을 갖춘 모듈
              상품등록, 상품삭제, 상품목록, 상품 재갱신 ... 

sell_.py : 판매관리의 기능을 갖춘 모듈
           카트에 담기, 카트 결제....

main_.py : 전체적으로 진행을 보여주는 모듈 C언어에서 main()와 비슷
